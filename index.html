Android Arabic Reader

Arabic Reader is a free and open source eBook reader. Supports epub, rtf, doc (MS Word), html, fb2(.zip), mobi (including non-protected kindle books), and plain text file formats. Arabic Reader is fast, highly customizable. Can use external TrueType/OpenType fonts and custom backgrounds. Screen brightness can be changed during reading (slide finger up/down along the left screen edge). Supports different day/night color schemes.

Based on FBReader, Arabic Reader fast and highly customizable - it can use external TrueType/OpenType fonts and custom backgrounds, the screen brightness can be adjusted while reading (slide finger up/down along the left screen edge), and different day/night color schemes can be selected.
